IndexExpandableListView
===========

Android IndexExpandableListView是在IndexableListView （https://github.com/woozzu/IndexableListView） 的基础上改进而来，
沿用了IndexableListView的索引快速定位方法和效果，加入了pinyin4j架包，实现了中文汉字拼音与英文字母的匹配。
